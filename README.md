# Coding Challenges Repository

## Overview

Welcome to the **Coding Challenges Repository**! This project contains a series of coding challenges designed to help developers practice and improve their problem-solving skills. The challenges are categorized into three difficulty levels: **Easy**, **Medium**, and **Hard**. Each challenge is accompanied by its solution and unit tests for validation.

This repository is perfect for anyone looking to improve their coding abilities, whether you're a beginner or an experienced programmer.

## Features

- **Coding Challenges**: A set of problems categorized by difficulty (Easy, Medium, Hard).
- **Solutions**: The solutions for each challenge to help you compare and improve your approach.
- **Unit Tests**: Each challenge includes test cases to validate the correctness of your solution.
- **Programming Languages**: Challenges and solutions are written in **Python** (you can adapt them to other languages).
